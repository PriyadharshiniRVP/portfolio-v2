# Priya's Portfolio

A personal portfolio website built using React and Vite.

## Features

- About Me section
- Experience section
- Projects showcase
- Responsive design (coming soon)
- Clean UI

## Tech Stack

- React
- JavaScript
- CSS
- Vite


## Installation

Clone the repository:

```bash
git clone <https://github.com/PriyadharshiniRVP/portfolio-v2>
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```
